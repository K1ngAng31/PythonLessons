Why PYTHON Programming?

Programming will help you foster creativity, problem solving, and reasoning. Programming will allow you to create something fun.

You will be able to turn nothng into something.

What I expect to contribute?

I want to be able to help you become a basic programmer. I want to help you create super cool programs. I want to go in depth and help you understand what you are doing without feeling overwhelmed.

I am no saying it will be easy, but it will be worth the struggle, and if you don't struggle, awesome, then there can be bigger things to move on to using PYTHON
